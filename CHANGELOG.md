# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-12

### Added
- Created the **Student Application Management System (SAMS)** Salesforce DX app.
- Created the **Inventory Management System** Salesforce DX app.
- Configured Lightning Web Components (`studentDashboard`, `applicationWizard`, `courseAvailability`, `homeDashboard`, `documentViewer`, `applicationStatusTracker`).
- Added Visualforce print-ready certificate `AdmissionPDF`.
- Added Apex controllers, triggers, trigger frameworks, and mock test coverage.
- Restructured workspace into an enterprise monorepo.
- Added unified root configurations for sfdx, eslint, prettier, husky, and npm.
- Added GitHub templates, contributing guides, security policies, and CI/CD workflow.
